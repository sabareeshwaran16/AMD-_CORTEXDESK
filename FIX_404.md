# 404 Error Fix

## Problem
Backend not running when you clicked START_DESKTOP.bat

## Solution

### Option 1: Try Again
1. Close all windows
2. Double-click `START_DESKTOP.bat` again
3. You should see a window titled "CortexDesk Backend"
4. Wait 5 seconds
5. Desktop app will open

### Option 2: Manual Start (Recommended)

**Terminal 1:**
```bash
python src\api.py
```
Wait for: `Uvicorn running on http://127.0.0.1:8001`

**Terminal 2:**
```bash
cd desktop
npm run dev
```
Wait for: `Local: http://localhost:5173`

**Browser will open automatically**

---

## Check Backend is Running

Open browser: http://localhost:8001

Should show:
```json
{
  "name": "Local AI Workspace Assistant",
  "version": "1.0.0",
  "status": "running"
}
```

If you see this, backend is working!
